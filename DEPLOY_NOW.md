## 🎯 第一步：创建 GitHub 仓库

### 访问 GitHub 创建仓库

👉 **https://github.com/new**

### 填写信息

```
Repository name: tcd-origin
Description: 跨文明古文字拓扑破译引擎 - TCD Origin
Public: ✓ 选择 (必须公开)
Initialize this repository with a README: ✗ 不勾选
```

点击 **Create repository**

---

## 🔗 第二步：推送代码

### 在终端执行以下命令

```bash
cd /workspace/projects

# 推送代码到 GitHub
git push -u origin main
```

### 输入 GitHub 用户名和密码/Token

```
Username: huangao2026
Password: 您的GitHub密码或Personal Access Token
```

> ⚠️ 如果您开启了双因素认证，需要使用 Personal Access Token 代替密码

## 🎉 第三步：验证推送成功

## 🚀 第四步：Railway 部署

### 访问 Railway

👉 **https://railway.app**

### 操作步骤

```
1. 点击 "Login with GitHub"
2. 授权 GitHub 账户
3. 点击 "New Project"
4. 选择 "Deploy from GitHub repo"
5. 在列表中找到 "tcd-origin"
6. 点击 "Deploy"
7. 等待 2-3 分钟
8. 点击生成的链接访问
```

## ⚠️ 常见问题

### Q: GitHub 推送失败？
```
原因1: 仓库已存在
解决: 先删除旧仓库或改名

原因2: 认证失败
解决: 使用 Personal Access Token
```

### Q: Railway 部署失败？
```
检查1: GitHub 仓库是否 Public（公开）
检查2: 代码是否完整上传
检查3: Railway 日志是否有报错
```
