## 一、拓扑古文字学概述

### 1.1 什么是拓扑古文字学

拓扑古文字学（Topological Paleography）是一门运用拓扑不变量对跨文明古文字进行结构比较的交叉学科。

**核心命题**：人类早期文字是否在图形结构层面存在跨文明的深层结构规律（deep structural regularities）？

### 1.2 核心思想：拓扑不变量比较而非语义比较

传统比较文字学（comparative graphology）依赖专家构建的语义对齐词库（expert-constructed semantic alignment lexicon）来建立跨文字的对应关系。

拓扑古文字学另辟蹊径：直接比较符号的几何-拓扑结构，不问"这个符号是什么意思"，而问"这个符号的结构长什么样"。

### 1.3 三层拓扑不变量层级互补体系

| 层级 | 名称 | 权重 | 功能 | 典型不变量 |
|------|------|------|------|-----------|
| L1 | 全局锚点 | 40% | 粗粒度快速筛选 | 对称性、宽高比 |
| L2 | 核心不变量 | 35% | 中粒度精确匹配 | 欧拉数、连通分量 |
| L3 | 局部指纹 | 25% | 细粒度验证 | 环数、像素密度 |

## 三、三大古文字体系

### 3.1 甲骨文（Oracle Bone Script）

| 属性 | 内容 |
|------|------|
| 年代 | 商代晚期，约公元前1200年—前1050年 |
| 地域 | 中国河南安阳殷墟 |
| 载体 | 龟甲、兽骨 |
| 已发现数量 | 约15万片甲骨，含约160万个字符 |
| 单字总量 | 约4600—5000个不同字符 |
| 已破译 | 约1200—1500个字符（约30%） |

**核心特点**：抽象派——刀刻硬骨形成方折线条

### 3.2 楔形文字（Cuneiform）

| 属性 | 内容 |
|------|------|
| 年代 | 苏美尔早期，约公元前3400年 |
| 地域 | 两河流域（美索不达米亚平原） |
| 载体 | 湿润泥板 |
| 符号总量 | 约1000—1500个 |
| 寿命 | 约3000余年 |

**核心特点**：工具派——芦苇杆压印形成楔形笔划

### 3.3 埃及圣书体（Egyptian Hieroglyphs）

| 属性 | 内容 |
|------|------|
| 年代 | 早王朝时期，约公元前3200年 |
| 地域 | 尼罗河流域 |
| 符号总量 | 约700—5000个 |
| 字体变体 | 碑铭体、僧侣体、大众体 |

**核心特点**：写实派——如精细绘画般描摹事物原貌
