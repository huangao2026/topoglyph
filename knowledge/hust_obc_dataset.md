# HUST-OBC 甲骨文识别数据集

## 数据集来源

**华中科技大学 (HUST)、华南理工大学、阿德莱德大学、安阳师范学院**

论文：[An open dataset for oracle bone script recognition and decipherment](https://arxiv.org/abs/2401.15365)

## 数据集规模

| 类别 | 字符数 | 图像数 |
|------|--------|--------|
| 已破译字符 | 1,588 | 77,064 |
| **未破译字符** | **9,411** | **62,989** |
| **总计** | **10,999** | **140,053** |

## 图像来源

- 书籍
- 网站
- 现有数据库

## 数据质量

- 所有图像和标签已由**甲骨文研究专家**审阅和校正
- 裁剪好、无释义、无水印
- AI破译专用

## 学术意义

这是迄今为止**最大的OBS识别和解密数据集之一**，旨在帮助破译未知的甲骨文字符。

## 在TCD Origin中的应用

作为**外部燃料**，用于：
- D1视觉特征层的形态捕捉
- D2拓扑几何层的压力测试
- 未破译字符的智能分析

---

## 使用规范

1. 仅使用数据集图像进行拓扑分析
2. 结合D1-D5推演系统进行深度解构
3. 禁止直接引用学术释义，以拓扑推理为准
4. 输出结果需包含置信度评估
