## 第一部分：产品性质与定位

### 1.1 产品性质

古文字拓扑破译智能体是一款基于人工智能和拓扑学的跨文明古文字分析平台。它通过多层次拓扑不变量分析和深度学习技术，实现对甲骨文、楔形文字、圣书体等古文字符号的智能识别、同源性分析和意义推断。

**核心属性**：
- **科研辅助工具**：为古文字研究提供量化分析支持
- **跨学科融合产品**：整合拓扑学、计算机视觉、历史语言学
- **智能推理引擎**：基于D1-D5五层架构的符号破译系统
- **知识库驱动**：集成火山引擎企业级知识库服务

### 1.2 目标用户

| 用户类型 | 使用场景 | 核心需求 |
|---------|---------|---------|
| 古文字研究者 | 甲骨文、楔形文字等古文字研究 | 量化分析、同源性判定 |
| 历史学家 | 跨文明文化交流研究 | 文化传播路径分析 |
| 考古学家 | 出土文物鉴定 | 符号年代判定、来源推断 |
| 语言学家 | 古文字演化研究 | 文字系统演化路径 |

## 第二部分：核心技术特征

### 2.1 D1-D5五层破译架构

这是本产品最核心的技术创新，采用五层递进式分析架构：

| 层级 | 名称 | 功能描述 | 技术基础 |
|------|------|---------|---------|
| **D1** | 视觉形态层 | 提取符号的视觉特征：笔画宽度、曲率、边缘密度 | 计算机视觉（CNN） |
| **D2** | 拓扑几何层 | 计算拓扑不变量：欧拉示性数、贝蒂数、环数、对称性 | 代数拓扑 |
| **D3** | 时间演化层 | 分析符号的历时演化路径和稳定性 | 历史语言学 |
| **D4** | 意义确权层 | 确定符号的语义场和语境锚定 | 维特根斯坦语言游戏理论 |
| **D5** | 逻辑坍缩层 | 多维度概率交叉验证，输出最终语义 | 贝叶斯推理 |

**架构特点**：
- **递进性**：从低层视觉特征到高层语义理解，逐层深入
- **互补性**：各层信息相互补充，提高准确率
- **自适应**：不同语义类型采用不同权重配置
- **可解释性**：每层分析结果可追溯

### 2.2 三层拓扑不变量体系

突破传统单一指标分析的局限，建立多层次拓扑特征体系：

| 层级 | 核心特征 | 权重 | 作用 |
|------|---------|------|------|
| **第一层：全局锚点层** | 对称性 | 50% | 跨文明同源性的基础锚点 |
| **第二层：核心不变量层** | 贝蒂数（B₀/B₁） | 30% | 拓扑结构的核心描述 |
| **第三层：局部指纹层** | 环数、欧拉特征数 | 20% | 文化特异性的精细区分 |

**创新点**：
- **对称性作为最强判别指标**：基于大规模数据分析，对称性相关系数达0.68（p<0.001）
- **语义类型自适应权重**：不同语义类型采用不同特征权重配置
- **多维度特征向量**：生成128维高维拓扑特征向量

### 2.3 跨文明符号分析能力

| 功能 | 描述 |
|------|------|
| 同源性判定 | 计算符号间的拓扑同源性距离 |
| 文化传播检测 | 区分独立起源与文化传播 |
| 认知趋同分析 | 识别人类对自然现象的共同认知 |
| 演化路径追踪 | 重建符号的历时演化路径 |

**关键发现**：
- **对称性**：最强跨文明同源性判别指标（统计显著）
- **环数**：区分独立起源与文化传播的辅助指标
- **特征区分力的概念依赖性**：不同语义类型需要不同特征

### 2.4 火山引擎知识库集成

- **海量知识存储**：支持结构化和非结构化数据
- **图文混合查询**：支持图像和文本的混合检索
- **多轮对话支持**：支持上下文感知的深度对话
- **实时更新**：知识库内容持续更新

### 3.2 功能详细说明

#### 3.2.1 拓扑特征提取

**功能描述**：
自动提取古文字符号的三层拓扑特征，生成完整的特征描述。

**输入**：
- 符号图像（支持JPG、PNG等格式）
- 或已提取的拓扑特征数据

**输出**：
```json
{
  "euler_characteristic": -2,
  "betti_numbers": [1, 3, 0],
  "ring_count": 1,
  "symmetry_score": 0.85,
  "feature_vector": [128维向量],
  "semantic_type": "天体类"
}
```

**技术亮点**：
- 自动识别语义类型
- 128维高维特征向量
- 完整的统计显著性评估

#### 3.2.2 跨文明同源性分析

**功能描述**：
对不同文明中的符号进行同源性分析，判断是否存在同源关系。

**拓扑同源性距离公式**：
```
D(S_a, S_b) = Σ ω_i × |T_i(a) - T_i(b)|
```

其中：
- ω_i：第i个拓扑特征的权重
- T_i(a)：符号a的第i个拓扑特征值
- T_i(b)：符号b的第i个拓扑特征值

**同源性等级划分**：
| 距离范围 | 同源性等级 | 解释 |
|---------|-----------|------|
| 0.00-0.20 | 高度同源 | 很可能同源或直接传承 |
| 0.20-0.40 | 中高度同源 | 存在一定关联 |
| 0.40-0.60 | 中度同源 | 可能存在间接关联 |
| 0.60-0.80 | 低度同源 | 独立起源可能性大 |
| 0.80-1.00 | 无显著同源 | 完全独立起源 |

#### 3.2.3 文化传播检测

**功能描述**：
区分符号间的相似性是来自文化传播还是独立起源的认知趋同。

**检测逻辑**：
```python
IF symmetry_similarity > 0.8 AND ring_difference == 0:
    -> 文化传播信号（强）
ELIF symmetry_similarity > 0.8 AND ring_difference > 0:
    -> 独立起源但认知趋同（弱）
ELIF symmetry_similarity > 0.6 AND symmetry_similarity <= 0.8:
    -> 可能存在部分同源性（中等）
ELSE:
    -> 独立起源
```

**关键指标**：
- **对称性相似度**：最强判别指标（p<0.001）
- **环数差异**：辅助判别指标

#### 3.2.4 D1-D5完整破译

**功能描述**：
对古文字符号进行从视觉到语义的完整分析。

**输出结构**：
```
D1 视觉形态层 → D2 拓扑几何层 → D3 时间演化层 
                                              ↓
D5 逻辑坍缩层 ← D4 意义确权层 ←──────────────
```

**每层输出**：
- **D1**：笔画特征、曲率特征、边缘密度
- **D2**：欧拉示性数、贝蒂数、环数、对称性
- **D3**：演化路径、稳定性评估、变异率
- **D4**：语义场分布、语境锚定
- **D5**：最终语义、概率分布、置信度

```bash
# 配置环境变量
export VOLCENGINE_API_KEY='your_api_key'
export VOLCENGINE_SERVICE_ID='your_service_id'
```

#### 基本使用

```python
from tools.tcd_origin_engine import TCDOriginEngine
from tools.tcd_origin_tools import tcd_full_analysis

# 方式一：直接使用引擎
engine = TCDOriginEngine()
result = engine.full_analysis(
    image_data="oracle_bone_sun.jpg",
    context="甲骨文祭祀场景",
    origin_estimate="甲骨文"
```
