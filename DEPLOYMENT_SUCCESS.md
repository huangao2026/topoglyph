## 🚀 Railway 部署（最后一步）

### 步骤 1：访问 Railway

👉 **https://railway.app**

### 步骤 2：登录

点击 **"Login with GitHub"**

### 步骤 3：创建项目

1. 点击 **"New Project"**
2. 选择 **"Deploy from GitHub repo"**
3. 在列表中找到 **"topoglyph"** 并点击
4. Railway 会自动检测并部署

### 步骤 4：等待部署

⏱️ 预计时间：**2-3分钟**

Railway 会自动：
- 拉取代码
- 安装依赖
- 构建镜像
- 启动服务

### 步骤 5：获取链接

部署成功后，Railway 会显示：

```
🌐 您的在线链接: https://xxxxx.railway.app
```

## 🆘 常见问题

### Q: Railway 显示部署失败？
A: 检查 GitHub 仓库是否有 `web_app.py` 和 `Dockerfile` 文件

### Q: 链接打不开？
A: 等待 1-2 分钟，服务启动需要时间

### Q: 如何自定义域名？
A: 在 Railway 项目设置中添加自定义域名
